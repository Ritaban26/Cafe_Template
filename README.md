# Cafe Template

A clean Next.js template for building cafe websites.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in your browser.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
